# Cash Flow Landlords – REIT Analysis

## Objective
Identify U.S. counties where rental investment is more profitable than flipping.

## Data Sources
- Zillow Home Value Index (ZHVI)
- Zillow Rent Index (ZORI)
- U.S. Census Median Income

## Methodology
- Built models to predict future rent (ZORI 2025)
- Compared:
  - Linear Regression
  - Random Forest
- Evaluated using R² and RMSE

## Results
- Best Model: Linear Regression (R² ≈ 0.97)
- Rental strategy outperforms flipping in most counties

## Key Insight
High rent yield markets provide better long-term returns than appreciation-based flipping.
